# ML_Stock_Trader
ML-driven stock trader using a bagged Random Tree ensemble trained on SMA ratio, Bollinger Bands %B, and MACD histogram indicators. Classifies daily signals (buy/sell/hold) via 3-day lookahead returns. Benchmarked against a manual rule-based strategy (using the same indicators mentioned) and buy-and-hold on in-sample and out-of-sample periods.
